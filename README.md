## 使用说明
1. 根目录配置到thinkphp-blog/下；
2. 后台登录密码默认为admin；
4. 若要允许百度爬虫抓取请把根目录下的robots.txt中的Disallow改为Allow；
5. 可以在用户管理中；将第三方账号标记为站长；然后后台就必须使用第三方账号登录以增强安全性；
6.新建一个数据库blog,将thinkphp-blog/data/blog.sql导入到数据库中
